# Prosty kalkulator

## Opis

Prosty kalkulator napisany w React.js przy użyciu klas. Pozwala na wykonywanie prostych obliczeń.

### Features

- Dodawanie
- Odejmowanie
- Mnożenie
- Dzielenie
- Oblicznie %

### Instalacja

1. Sklonuj to repozytorium

2. cd react-calc

3. npm install

4. npm start - żeby zaczął działać
